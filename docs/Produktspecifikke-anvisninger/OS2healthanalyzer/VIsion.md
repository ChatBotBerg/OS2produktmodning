# OS2healthanalyzer Vision, mission og målsætning

VISION: OS2 Healthanalyzeren skaber grundlaget for evaluering af modenhed i danske Open sorurce øsninger 

Mission: at skabe en defactostandard for modenheds målign i en dansk kontekst
Os2 Healthanalyzer måler Modenhed af Open Source produkter ud fra et sæt af givne parametre

Os2 Healthanalyzer udvikles i samarbejde med KL og DIGST

Os2 Healthanalyzer kan
- foretage Automatiserede målinger på Open-Source produkter
- agere som Data feed for
  -  KL's rapport om 10 genbrugelige løsninger
  -  OS2's interne modenhedsmålinger
  -  Digst katalog over Danske Opensource produkter
  -  EU's katalog over Europæiske Open source produkter

Kildekoden vedligeholdes og videreudvikles i OS2 regi
Driftinstansen for ejes og financieres af DIGST 
- Skabe en dansk defactoløsning for automatiserede løsninger

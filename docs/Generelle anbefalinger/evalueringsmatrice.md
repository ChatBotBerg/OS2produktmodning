# Prioriterings Matrix

**Vurderingsskala for Prioriteringsmatrix (1-5):**
*   **1**: Meget lav/mindre vigtig
*   **2**: Lav/mindre prioritet
*   **3**: Moderat
*   **4**: Høj/vigtig
*   **5**: Meget høj/kritisk

**Definition af skala for hvert kriterie (eksempler):**

*   **Forretningsværdi?**
    *   **1**: Minimal forretningsværdi.
    *   **5**: Essentiel forretningsværdi, afgørende for organisationens mål.
*   **Impact-timer pr. uge, hvis ikke løst?**
    *   **1**: Ingen mærkbar tidsspilde/impact.
    *   **5**: Stort tidsspilde (>10 timer/uge) for mange brugere, hvis ikke løst.
*   **Hard wins?**
    *   **1**: Ingen direkte "hard wins" (målbare besparelser/indtægter).
    *   **5**: Stor og direkte målbar positiv effekt på økonomi/produktivitet.
*   **Soft wins?**
    *   **1**: Ingen mærkbare "soft wins" (forbedret brugertilfredshed, omdømme).
    *   **5**: Stor og bred positiv effekt på brugertilfredshed, medarbejderengagement eller eksternt omdømme.
*   **Tidskritisk?**
    *   **1**: Ingen tidskritiske deadlines, kan udskydes.
    *   **5**: Meget tidskritisk, skal løses omgående pga. deadline/lovkrav.
*   **Strategisk vigtighed?**
    *   **1**: Mindre strategisk betydning.
    *   **5**: Afgørende for organisationens langsigtede strategi og mål.

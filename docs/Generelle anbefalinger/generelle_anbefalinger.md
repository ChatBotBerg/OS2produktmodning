# Generelle anbefalinger og skabeloner

Denne sektion indeholder anvisninger og skabeloner til brug i den daglige produktforvaltning. 

Alt hvad du finder her er baseret på standard metoder og best practice fra IT-Industrien.

**Anbefalingerne**  fokusere på hvordan man i praksis kan anvende kommercielle styringsmodeller i en  Open-Source kontekst. 


**Skabelonerne** er eksempler på hvordan man let og enkelt kan implementerer kendte styringsværktøjer i den daglige forvaltning af Open-Source produkter. De er er alle baseret på markdown og mermaid standardfunktionalitet og er til fri afbenyttelse for hvem der måtte have glæde af dem. 

**Råfilerne** bag dette website finder du i i Github https://chatbotberg.github.io/OS2produktmodning/
